## What is Utility Belt?

Utility Belt is my personal collection of practical tools and micro-applications.

Every tool is:

- Lightweight
- Browser-based
- Mobile friendly
- Free to use
- Zero installation
- Built with HTML, CSS and JavaScript

The goal is simple:

> Create useful things that solve real problems.
